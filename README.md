# CSharp WinForms Projects

This repository contains simple C# Windows Forms projects created while learning desktop application development.

## Projects

### 1. Pizza Order App

A Windows Forms application that allows the user to customize and order a pizza.

#### Features

* Select pizza size
* Select toppings
* Select crust type
* Choose eat-in or take-out
* Calculate total price automatically
* Show order summary
* Confirm or reset the order

#### Concepts Used

* Windows Forms controls
* Radio buttons
* Checkboxes
* Group boxes
* Labels
* Buttons
* Event handling
* Basic price calculation
* Form reset logic

---

### 2. Tic-Tac-Toe Game

A Windows Forms Tic-Tac-Toe game for two players.

#### Features

* Two-player gameplay
* Player turn switching
* Winner detection
* Draw detection
* Restart game option
* Images for X, O, and empty cells
* Highlight winning line

#### Concepts Used

* Windows Forms
* Buttons
* Images and resources
* Enums
* Structs
* Event handling
* Game logic
* Drawing custom lines using Paint event

## Technologies Used

* C#
* .NET Framework
* Windows Forms

## Purpose

These projects were built to practice C# desktop application development, user interface design, event-driven programming, and basic application logic.
