# CSharp WinForms Projects

This repository contains simple C# Windows Forms projects created while learning desktop application development, event-driven programming, custom controls, and basic application logic.

## Projects

### 1. Pizza Order App

A Windows Forms application that allows the user to customize and order a pizza.

#### Features

* Select pizza size
* Select toppings
* Select crust type
* Choose eat-in or take-out
* Calculate total price automatically
* Show order summary
* Confirm or reset the order

#### Concepts Used

* Windows Forms controls
* Radio buttons
* Checkboxes
* Group boxes
* Labels
* Buttons
* Event handling
* Basic price calculation
* Form reset logic

---

### 2. Tic-Tac-Toe Game

A Windows Forms Tic-Tac-Toe game for two players.

#### Features

* Two-player gameplay
* Player turn switching
* Winner detection
* Draw detection
* Restart game option
* Images for X, O, and empty cells
* Highlight winning line

#### Concepts Used

* Windows Forms
* Buttons
* Images and resources
* Enums
* Structs
* Event handling
* Game logic
* Drawing custom lines using the Paint event

---

### 3. Traffic Light Control

A Windows Forms project that simulates a traffic light using a custom UserControl.

#### Features

* Red, orange, and green light switching
* Countdown timer
* Custom UserControl
* Custom events for light changes
* Configurable red, orange, and green durations
* Image resources for each light state

#### Concepts Used

* Windows Forms
* UserControl
* Timer
* Events
* Custom EventArgs
* Properties
* Resources
* State switching logic

---

### 4. Pool Table Management

A Windows Forms project for managing pool table sessions and calculating usage fees.

#### Features

* Multiple pool tables
* Start and stop timer for each table
* End table session
* Calculate total fees based on hourly rate
* Custom UserControl
* Custom event when a table session is completed
* Editable player name using context menu

#### Concepts Used

* Windows Forms
* UserControl
* Timer
* Events
* Custom EventArgs
* ContextMenuStrip
* Custom properties
* Fee calculation logic

## Technologies Used

* C#

* Visual Studio

## How to Run

1. Download or clone the repository.

```bash
git clone https://github.com/tareq-mujahedd/CSharp-WinForms-Projects.git
```

2. Open the project folder you want to run.

3. Open the `.sln` file using Visual Studio.

4. Make sure the `.NET desktop development` workload is installed.

5. Press `F5` or click `Start` to run the application.

## Requirements

* Windows
* Visual Studio
* .NET Framework
* .NET desktop development workload

## Purpose

These projects were built to practice C# desktop application development, user interface design, event-driven programming, custom UserControls, timers, events, and basic application logic.
